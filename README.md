# README와 버전관리

## I.README (Read Me)
### 1. README 란 ?
  완성된 프로그램의 설명서
  구현 중인 프로젝트의 현황을 정리한 문서


### 2. Markdown이란?
  일반 사용자도 웹에서 쉽게 글을 쓸 수 있도록 만든 문법이다.\
  일반 텍스트를 웹 기술로 변환해 준다.

### 3. Markdown의 주요 목적
  읽기 쉽고 쓰기 쉽게 만드는 것
